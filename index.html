  • Decision Tree from Scratch (without using any Machine Learning Libraries)
    Detailed Explanation is on the notes of the python file
    • Python Model .py file   

      The first graph demonstrates the framework of a decision tree:
      1. The algorithm first goes through all of the column features

      2. Within each of the column, the algorithm sorts the rows/observations in accending order of the column value.

      3. Then it treats each of the observation as a split point which seprates obervations; obervations that are less than the current split point are placed into the left group; and those greater than the current split point are placed into the right group
